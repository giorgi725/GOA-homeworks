Backend Development

Backend არის ვებსაიტის ან აპლიკაციის “შიდა ნაწილი”, რომელიც მუშაობს სერვერზე და პასუხისმგებელია მონაცემების დამუშავებაზე, შენახვაზე და მომხმარებლის მოთხოვნებზე პასუხის გაცემაზე. თუ Frontend არის ის, რასაც მომხმარებელი ხედავს, Backend არის სისტემა, რომელიც ყველაფერს ამუშავებს ფონში.

Server

სერვერი (Server) არის კომპიუტერი ან პროგრამა, რომელიც იღებს მომხმარებლის მოთხოვნებს და აბრუნებს პასუხს.

მაგალითად:როდესაც მომხმარებელი შედის სოციალურ ქსელში,აგზავნის შეტყობინებას

Database

მონაცემთა ბაზა (Database) არის ადგილი, სადაც ინფორმაცია ინახება.

მაგალითად: მომხმარებლების ანგარიშები, პაროლები, პოსტები


API

API (Application Programming Interface) არის სისტემა, რომლის საშუალებითაც Frontend და Backend ერთმანეთთან ურთიერთობენ.

მაგალითად: Frontend აგზავნის მოთხოვნას API-ზე, Backend ამოწმებს მონაცემებს

Backend Stack-ები

Stack ნიშნავს ტექნოლოგიების კომბინაციას, რომლითაც იქმნება პროექტი.
