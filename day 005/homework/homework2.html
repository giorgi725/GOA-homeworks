browser არის ერთ-ერთი ყველაზე საჭირო პროგრამა თანამედროვე ტექნოლოგიებში. მისი მეშვეობით ჩვენ შეგვიძლია რამოდენიმე წამში გავიგოთ ნებისმიერი ინფორმაცია მთელ მსოფლიოს მაშტაბით.
browser ასრულებს რამდენიმე მთავარ ფუნქციას ერთდრუოლად:მთარგმნელი(HTML,CSS,JS), მაკონტროლებელი(ზრუნავს ჩვენს უსაფრთხოებაზე,იცავს ჩვენს მონაცემებს) და დამაკავშირებელი(აერთებს ჩვენს კომპიუტერს ინტერნეტს და აგზავნის მოთხოვნებს სერვერებზე რათა გვაჩვენოს საჭირო ვებსაიტები).

HTML ს იყენებს საიტის შესაქმნელად(ჩონჩხი)
CSS ს იყენებს დიზაინისთვის(ამატებს ფერებს,ზომებს და სხვა)
JavaScript ის დახმარებით უკვე ფუნქციებს აძლევს(ანიმაციებს,ილაკების რეაგირებ და სხვა).
